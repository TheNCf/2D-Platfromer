# 2D-Platfromer

## Controls
| Action | Control |
| ------------- | --------------------------- |
| Movement      | <kbd>WASD</kbd>             |
| Jump          | <kbd>Spacebar</kbd>         |
| Slide         | <kbd>Shift</kbd> (grounded) |
| Dash          | <kbd>Shift</kbd> (in air)   |
| Pick up / Use | <kbd>E</kbd>                |
| Climb up      | <kbd>W</kbd>                |
| Melee attack  | <kbd>LMB</kbd>              |
